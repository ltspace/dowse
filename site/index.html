mcp --read-only

你的 AI 读过整个互联网,却读不到你的硬盘。dowse 内置只读 MCP server,Claude / Cursor 等客户端接入后可直接检索你索引过的本地文件。

  • search全文搜索,按相关度排序[RO]
  • preview命中处 ~1500 字长上下文[RO]
  • index_status索引概况[RO]
