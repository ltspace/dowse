Dowse（问渠）是一款本地全文搜索工具。用一个快捷键搜索文件名、PDF 与 Office 正文、源代码，以及截图里的文字。

下载 Windows 版 查看源码

搜索文件内容，不只搜索文件名

Dowse 会提取并索引纯文本、Markdown、源代码、PDF、DOCX、XLSX 和 PPTX 的正文。在 Windows 上，它还使用系统自带的 Windows.Media.Ocr，让 PNG、JPG、WebP 与 BMP 图片中的文字也可以被搜索。

搜索基于 tantivy 的持久倒排索引与 BM25 排序。中文使用 jieba 分词，不会被粗暴拆成任意字串；旧的 GBK 文本也会在索引前自动识别。解析、索引、OCR 和查询全部在本机完成，没有云端解析器，也没有遥测。

一个快捷键，回到资料的上下文

按 Alt+` 呼出搜索浮窗，输入记得的一句话，用方向键选择结果，再按 Enter 打开。结果会展示命中片段与实时预览；还可以按文档、代码、图片筛选，并按相关性、修改时间或文件大小排序。

Dowse 与其他 Windows 搜索工具有什么不同

工具主要方式适合场景
Dowse持久索引文件名与正文，并加入图片 OCR频繁搜索文档、代码、中文资料和截图文字
Everything文件名索引极快；内容查询通常按需扫描文件即时查找文件名，偶尔在限定范围内搜索内容
Windows Search系统内置索引，依赖位置配置与内容筛选器不安装第三方工具，搜索标准 Windows 文档目录
dnGrep按需扫描文本，支持正则表达式针对特定目录做正则搜索与批量替换

让 AI 只读搜索你的本地资料

dowse mcp 提供三个只读 MCP 工具：排序后的全文搜索、命中位置附近的长预览，以及索引状态。Claude Code、Cursor 和其他 MCP 客户端可以搜索你已建立索引的文件夹，而不必把硬盘访问权限交给远程服务。

Claude Code · 一行接入
claude mcp add --scope user dowse -- dowse mcp
任意 MCP 客户端 · JSON 配置
{
  "mcpServers": {
    "dowse": {
      "command": "dowse",
      "args": ["mcp"]
    }
  }
}

安装 Dowse

  1. 从 GitHub Releases 下载最新的 Windows 安装包。
  2. 选择需要建立索引的文件夹。管理员权限不是必需条件，只会在可用时解锁 NTFS MFT / USN 快速路径。
  3. 按 Alt+`，输入文档正文或截图里记得的一句话。

安装了 Rust 1.85 或更高版本的 CLI 用户，也可以运行 cargo install dowse。

常见问题

Dowse 会上传我的文件或搜索词吗？

不会。文件解析、索引、OCR 与查询都在本机完成，应用不包含遥测。本地索引含有提取出的文本，仍应像源文件一样受到保护。

Dowse 能搜索扫描版 PDF 吗？

Dowse 会提取 PDF 已有的文本层。当前 OCR 管线用于图片文件，尚不会把纯图片扫描 PDF 的每一页转换成可搜索文字。

Dowse 必须用管理员权限运行吗？

不需要。普通索引和搜索可在标准用户权限下运行；管理员权限只用于条件允许时启用更快的 NTFS MFT 与 USN Journal 路径。

Dowse 是开源软件吗？

是。桌面端、CLI、搜索库与 MCP server 均以 MIT 或 Apache-2.0 双许可证开源。
